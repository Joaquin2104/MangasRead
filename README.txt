Tienda de Mangas

Descripción del proyecto

El proyecto consiste en una página web destinada a la compra de mangas. 
La plataforma está pensada para usuarios particulares que desean adquirir mangas de manera individual, por lo que no cuenta con funcionalidades de compra mayorista.
